Null Space of a Matrix \(A\)

Say that shape of matrix \(A\) is \(m\times n\) , \(A\in\mathbb{R}^{m\times n}\).

▶  For a matrix say \(A\) the Null space is the space of all \(\vec{x}\) that solves \(A\vec{x}=\vec{0}\).
▶  Null space is a vectors space.

There might be some dependent column vectors in matrix \(A\), if there are some dependent column vectors then,
▶  Linear combinations of those dependent column vectors gives us the Null Space of Matrix \(A\)
▶  If there are \(k\) dependent column vectors then our Null Space \(\in\mathbb{R}^k\).
▶  If there is not any dependent column vectors then the Null Space is just the \(\vec{0}\)(zero vector), because \(\vec{x}=\vec{0}\) solves \(A\vec{x}=\vec{0}\).
