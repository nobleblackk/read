Rank of a \(m\times n\) matrix \(A\) of rank \(r\)

A rank is the number of pivots of matrix \(A\).
So we have \(n\) column vectors in \(\mathbb{R}^m\), and there are \((n-r)\) (dependent)column vectors which we can get by the linear combination of other \(r\) (independent)column vectors.
\(\bullet\)  # pivots can't be \(\gt\) # rows (\(m\)), so \(r\leq m\)
\(\bullet\)  # pivots can't be \(\gt\) # columns (\(n\)), so \(r\leq n\)
